# yanolja-whitelabel-front

## Tech Stack
- Nuxt^2.0.0
- vuex(상태관리)
- scss(css 전처리)

## 실행방법
```
install dependencies
$ npm install

serve with hot reload at localhost:3000
$ npm run dev

build for production and launch server
$ npm run build
$ npm run start

generate static project
$ npm run generate
```

## 이렇게 개발 합시다!
> 변화의 유연한 코드 작성을 위해 지향하는 작지만 강력한 원칙들

- 관심사의 분리(SOC)
- 비즈니스 로직과 뷰 로직의 분리
- SOLID 원칙
    - 단일 책임 뭔칙(SRP)
    - 개방/폐쇄 원칙(OCP)
    - 리스코프 치환 원칙(LSP)
    - 인터페이스 분리 원칙(ISP)
    - 의존성 역전 원칙(DIP)
- 추상화하기 어렵다면 차라리 분리하기
- 길더라도 명확한 이름 짖기
